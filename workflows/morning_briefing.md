# 🌅 아침 브리핑 자동화 워크플로우

> Claude Desktop에서 바로 복사-붙여넣기하여 사용하세요.

---

## 📋 기본 아침 브리핑 프롬프트

아래 텍스트를 Claude Desktop 대화창에 붙여넣으세요:

```
오늘 하루를 준비하는 아침 브리핑을 해줘.

1. Google Calendar에서 오늘 일정 모두 확인해서 시간순으로 알려줘
2. Gmail에서 어제 저녁 이후 받은 읽지 않은 메일 중 답장이 필요한 것 요약해줘
3. Notion에서 "오늘 할 일" 페이지 있으면 불러와줘. 없으면 새로 만들어줘
4. 오늘 집중해야 할 우선순위 3가지를 정리해서 Notion 페이지 상단에 추가해줘
```

---

## 🔄 심화 버전 (주간 계획 연동)

```
월요일 아침 주간 계획 브리핑:

1. Google Calendar에서 이번 주 전체 일정 가져와줘
2. Gmail에서 지난 주말 동안 온 중요 메일 정리해줘
   - 중요도: 높음/중간/낮음으로 분류
   - 각 메일별 필요 액션 한 줄 요약
3. Notion에 "2026-W{주차} 주간계획" 페이지 만들어줘:
   - 이번 주 주요 목표 3개
   - 요일별 주요 일정
   - 주간 KPI 트래킹 테이블
4. 이번 주 가장 중요한 미팅/마감 3가지 알려줘
```

## ⚡ 빠른 15초 브리핑

```
오늘 캘린더 일정이랑 중요 메일 5줄 요약해줘.
```

## 📊 업무 대시보드 생성

```
Notion에 오늘 날짜로 일일 업무 대시보드 만들어줘.

포함 내용:
- 오늘 날짜와 요일
- Google Calendar: 오늘 미팅/일정 목록
- Gmail: 미처리 중요 메일 (링크 포함)
- 오늘 TODO 체크리스트 (빈 항목 5개)
- 오늘의 집중 목표 (1개)
- 메모 섹션

형식: Notion 페이지로 저장, 제목은 "📅 {오늘날짜} 일일계획"
```

## 🤖 자동화 팁

### Claude Desktop에서 반복 사용하기
1. 위 프롬프트를 텍스트 파일로 저장
2. 매일 아침 파일 열어서 복사
3. Claude Desktop에 붙여넣기 → 실행

### 더 스마트하게 — 프롬프트 개인화
자신의 상황에 맞게 다음 부분을 수정하세요:
- `"오늘 할 일"` → 실제 Notion 페이지/데이터베이스 이름으로 변경
- 중요 메일 필터 조건 추가 (예: `보낸사람에 boss@company.com 포함된 것만`)
- 캘린더 필터 추가 (예: `개인 캘린더 제외하고 업무 캘린더만`)
