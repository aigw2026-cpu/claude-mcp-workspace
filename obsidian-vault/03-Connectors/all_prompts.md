## 🔀 멀티 커넥터 통합 프롬프트

### 이메일 → Notion 자동 정리
```
Gmail에서 오늘 온 중요 메일을 읽고,
Notion의 "받은업무" 데이터베이스에 각각 추가해줘.
필드: 제목, 보낸사람, 요약, 우선순위(상/중/하), 마감일
```

### 일정 → 노트 자동화
```
Google Calendar에서 오늘 미팅 목록 가져와서,
각 미팅별로 Notion에 회의록 템플릿 페이지 만들어줘.
템플릿: 참석자, 안건, 결정사항, 다음 액션 섹션 포함
```

### 주간 리포트 자동화
```
이번 주를 돌아보는 주간 리포트를 만들어줘:
1. Calendar에서 이번 주 참석한 미팅/이벤트 집계
2. Gmail에서 주고받은 중요 메일 요약
3. Notion에서 완료한 TODO 항목 확인
4. 위 내용을 Notion에 "주간리포트-{날짜}" 페이지로 저장
```

## 💡 프롬프트 작성 팁

1. **구체적으로 요청** - "메일 정리해줘" < "오늘 오후 2시 이후 받은 미처리 메일 우선순위 순으로 정리"
2. **출력 형식 지정** - "테이블 형식으로", "번호 목록으로", "Notion 페이지로 저장"
3. **조건 명시** - "답장 필요한 것만", "중요도 높은 것만", "이번 달 데이터만"
4. **여러 서비스 연결** - Gmail 읽기 + Notion 저장을 한 번에 요청 가능
